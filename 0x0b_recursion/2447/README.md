# 2447번. 별 찍기 - 10

## 문제
https://www.acmicpc.net/problem/2447

---

## Key Points

1. 9x9는 3x3이 8개에 가운데 비어있는 구조, 27x27 또한 9x9 8개에 가운데가 비어있는 구조이다.

2. 이를 재귀적으로 생각하면 3^(k+1)x3^(k+1)은 3^kx3^k 의 각 row를 3배하여 3번 복사하면 된다.

    - 가운데 비어있는 구조를 만들기 위해, 2번째로 복사할 때는 `row * 3`하는 것이 아니라, `row + " " * k + row` 해줘야 한다.
