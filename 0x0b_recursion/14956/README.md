# 14956번. Philosopher’s Walk

## 문제
https://www.acmicpc.net/problem/14956

---

## Key Points

1. 정사각형을 4등분하면 전 단계(`prev_n = n // 2`)의 모양이 나오는 재귀 문제.

    - `m`을 `prev_n ** 2`로 나눴을 때 몫과 나머지가 각각 위치한 `quadrant`(사분면)과 전 단계의 `m`이 된다.
        - 0사분면: 왼쪽 아래, 1사분면: 왼쪽 위, 2사분면: 오른쪽 위, 3사분면: 오른쪽 아래

    - 1, 2사분면의 경우 전 단계와 모양이 같지만, 0사분면은 시계 방향으로 90도 회전되고 3사분면은 반시계 방향으로 90도 회전된다.
        - 순서의 관점에서 보면, 0사분면: $y = x$ 축 기준으로 뒤집힘, 3사분면: $y = -x + prev_n$ 축을 기준으로 뒤집힘.

    - 전 단계에서의 좌표를 회전한 값을, 사분면에 따라 위치 보정(shift) 해준다.
        - 0사분면: 위치 보정 없음
        - 1사분면: y += prev_n
        - 2사분면: x, y 모두 += prev_n
        - 3사분면: x += prev_n
    
    - 정리하자면, 전 단계에서의 해당 위치 좌표 (x, y)를 구하고, 사분면에 따라 회전한 다음, 사분면에 따라 위치를 보정해주면 된다.

2. 문제는 1-based이지만, 나머지 때문에 0-based로 시작하는게 편해서, 최초 인자를 `m-1`로 주었다. (`based_coord`를 1-based로 하여 보정)
