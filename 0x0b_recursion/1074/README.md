# 1074번. Z

## 문제
https://www.acmicpc.net/problem/1074

---

## Key Points

1. 2^n * 2^n 배열을 4등분 해보면 4사분면 위치의 값에 `(2 * (r // 2^(n-1)) + (c // 2^(n-1))) * 2^(n-1) * 2^(n-1)`를 더해주면 되는 것을 알 수 있다.

    - 수식으로 표현되어 어려워 보일 수 있지만, (1, 1)은 (0, 0)의 값에서 1를 3번 더해주고, (3, 3)는 (1, 1)의 값에서 4를 3번 더해주면 된다. (1 -> 4 -> 16 -> ...)

      - 즉, `(2 * (r // 2^(n-1)) + (c // 2^(n-1)))` 는 Z의 0, 1, 2, 3 중에 어디에 해당되는지 offset 을 찾는 것이다.

      - `2^(n-1) * 2^(n-1)`은 몇을 더해줄지 구하는 것이다. (1 -> 4 -> 16 -> ...)

    - 2x2 배열에서 (1,1)의 값은 4사분면 위치(0,0)의 값인 0 + ((2 * 1) + 1) = 3

    - 4X4 배열에서 (3,3)의 값은 4사분면 위치(1,1)의 값인 3 + ((2 * 1) + 1) * 2 * 2 = 12

2. 위 특성을 활용하여 재귀를 구현하면 된다.

    - (3, 3) 값 구하려면 => (1, 1) 값 필요. 이 값 구하려면 => (0, 0) 값 구하기
