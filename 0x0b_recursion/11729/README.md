# 11729번. 하노이 탑 이동 순서

## 문제
https://www.acmicpc.net/problem/11729

---

## Key Points

1. 대표적인 재귀 문제인 하노이탑 문제이다.

2. n개의 원판을 1에서 3번 장대로 옮기려면 다음과 같은 작업을 수행하면 된다. (귀납적 방법)

   - (1) n-1개 원판을 1번에서 2번으로 옮긴다.

   - (2) n번째 원판을 3번으로 옮긴다.

   - (3) 2번으로 옮긴 n-1개 원판을 3번 원판으로 옮긴다. 

3. 위 과정을 코드로 구현하면 다음과 같다.

    ```python
    move(1, 2, n-1)  # (1)
    path.append("1 -> 3")  # (2)
    move(2, 3, n-1)  # (3)
    ```

4. (1)과 같이 1번에서 시작하는 경우는 2번 장대에 임시로 옮겨놔야하고, 2번에서 시작하는 경우는 1번 장대에 임시로 옮겨놔야 한다.

    - solve.py 풀이에서는 장대를 set을 활용하여 `POLES = {1, 2, 3}`로 정의해두고, `remain = (POLES - {src, dest}).pop()`로 임시로 옮겨 놓을 수 있게 해두었다.

        ```python
        remain = (POLES - {src, dest}).pop()
        move(src, remain, n-1)
        path.add(f"{src} -> {dest}")
        move(remain, dest, n-1)
        ```

    - 장대가 3개의 숫자로 되어 있으니 `6 - src - dest`를 해도 같은 기능을 구현할 수 있다. (더 효율적임)
