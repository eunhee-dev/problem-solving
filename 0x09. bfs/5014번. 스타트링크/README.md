# 5014번. 스타트링크

## 문제
https://www.acmicpc.net/problem/5014

---

## Key Points

1. [1697번. 숨바꼭질](../1697번.%20숨바꼭질/) 문제와 동일

    - 숨바꼭질 문제에서는 100,000을 넘어가는 경우도 (원래는) 고려해야 하지만, 해당 문제는 엘레베이터 층을 넘어갈 수 없어 고려 X

    - 숨바꼭질 문제는 +1, -1이 있어 무조건 도달할 수 있지만, 이 문제는 u, d 값에 따라 도달하지 못할 수 있음

2. 큐에서 pop할 때 `x == g - 1` 을 체크해주면 도달한 경우 바로 return 할 수 있음 (효율적)

    - 층이 1부터 시작하기 때문에, s와 g를 -1 해주어 체크해야 배열의 인덱스와 맞음
