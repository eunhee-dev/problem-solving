# 2583번. 영역 구하기

## 문제
https://www.acmicpc.net/problem/2583

---

## Key Points

1. board를 구성하는 것 외에는 기존 bfs 문제들과 개념적으로 동일하다. ([1926번. 그림](../1926번.%20그림/))

2. 지금까지 행을 x, 열을 y로 표현하였지만 문제에서의 (x, y)는 직교 좌표계로 가로: x, 세로 : y여서 헷갈릴 수 있다.

    - 이 문제에서도 행을 계산할 때 y가 나와서 헷갈릴 수 있음
  
        ```python
        for x1, y1, x2, y2 in rect_coord:
            for x in range(y1, y2):
                for y in range(x1, x2):
                    board[x][y] = 0
        ```

    - (x, y) 대신 (r, c) 또는 (i, j) 사용하는 방식을 고민해보자.

3. 문제와 다르게 뒤집어서 board를 구성해도 답은 동일하다.

    - 문제에서는 왼쪽 아래를 (0, 0)으로 설정하였지만, 우리가 흔히 사용하는 list는 (개념적으로) 왼쪽 위가 (0, 0)이다.

    - 문제 그대로 구현한다면 `range(m - y2, m - y1)`로 해야하지만, 이 문제에서는 개수와 크기만 구하기 때문에 `range(y1, y2)` 해도 답은 같다.
