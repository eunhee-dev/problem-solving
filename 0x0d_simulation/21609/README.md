# 21609번. 상어 중학교

## 문제
https://www.acmicpc.net/problem/21609

---

## Key Points

1. 이 문제는 다음과 같이 나누어 생각하면 좋다.

    - 크기가 가장 큰 블록 그룹 찾기
        - 점수 계산하기
        - 크기가 가장 큰 블록 그룹 지우기

    - 중력 적용하기

    - 반시계 방향으로 회전하기

2. 문제 조건을 잘 읽어야하는 문제이다.

    - 크기가 가장 큰 블록 그룹 조건 순서:
        1) 크기가 가장 큰 블록
        2) 무지개 블록이 가장 많은 블록 그룹
        3) 기준 블록 행이 가장 **큰** 것
        4) 기준 블록 열이 가장 **큰** 것

    - 기준 블록은 **무지개 블록이 아닌** 블록 중에서 행의 번호가 가장 **작은** 블록, 그러한 블록이 여러개면 열의 번호가 가장 **작은** 블록

    - 즉, 기준 블록 자체는 그룹 내 행/열이 가장 작은 일반 블록이지만, 기준 블록의 위치가 가장 **큰** 그룹을 선택해야 한다. (나는 기준 블록 행과 열이 가장 작은 블록 그룹을 골라서 한참 헤맸다...)

3. 크기가 가장 큰 블록 그룹은 bfs를 활용하여 찾아주면 된다.

    - 여기서 중요한 포인트는, 무지개 블록의 경우 bfs가 끝난 이후에 visited 배열에 방문한 것으로 표시되어 있으면 안된다는 것이다.
        - 왜냐하면 무지개 블록은 다른 블록들과 공유할 수 있기 때문이다.
        - 잘 구분해야 하는 것이, bfs가 끝나기 전까지는 중복 방문을 피하기 위해 visited 배열에 방문한 것으로 표시해야 한다. (bfs가 끝나고 나서 되돌려야 한다.)
        - 이를 해결하기 위해, (1) 무지개 블록을 기록하고 bfs가 끝난 후 visited 배열에 `False`로 돌려놓는 방법, (2) visited 배열에 블록 번호를 적어놓는 방법 등이 있다.
        - 나는 단순하게 (1)번 방법을 사용했는데, 무지개 블록을 따로 기록하지 않아도 된다는 점에서 (2)번 방법이 더 효율적일 수 있을 것 같다.

    - 그리고, 크기가 가장 큰 블록 그룹 조건을 순서대로 잘 따져봐야 한다.
        - bfs의 시작점이 행이 커지는 순 -> 열이 커지는 순으로 진행되고, 무지개 블록보다 큰 블록만 시작점으로 삼기 때문에, 뒷 순서에 나온 그룹을 채택하면 자연스럽게 뒤에 2개 조건은 달성된다.

            ```python
                for x in range(n):
                    for y in range(n):
                        if board[x][y] > RAINBOW_BLOCK and not visited[x][y]:
                            bfs(x, y)
            ```

        - 아래 코드와 같이 크기가 가장 큰 블록일 경우, 크기가 같으면 무지개 블록이 많거나 같은 경우 크기가 가장 큰 블록 그룹을 갱신한다. (무지개 블록 개수가 같으면, 뒤에 나올 수록 기준 블록의 행과 열이 더 크기 때문에 같은 경우도 갱신)

            ```python
            # 1. 새 그룹의 크기가 더 크면 무조건 갱신
            if (len(curr_group) > len(largest_group)
                # 2. 크기가 같다면,
                or (len(curr_group) == len(largest_group)
                    # 3. 무지개 블록 수가 더 많거나 같을 때 갱신
                    # (같을 때도 갱신해야 나중에 찾은 그룹, 즉 기준 블록 행/열이 더 큰 그룹이 선택됨)
                    and len(rainbow_blocks) >= largest_rainbow_cnt)):
                largest_group = curr_group
                largest_rainbow_cnt = len(rainbow_blocks)
            ```
