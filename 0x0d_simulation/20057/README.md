# 20057번. 마법사 상어와 토네이도

## 문제
https://www.acmicpc.net/problem/20057

---

## Key Points

1. 문제에서 구현하라는대로 구현하면 되는 문제이다.

    - 토네이도 구현하기
        - 서, 남, 동, 북 순으로 방향이 바뀜
        - 크기는 두 번 방향이 바뀔 때마다 1씩 증가함

    - 토네이도에 따라 움직인 모레 좌표 구하기
        - 좌표가 격자 밖으로 나갔으면 정답에 더해주기
        - 안에 있으면, 격자에 반영하기 
    

2. 모레의 이동을 나타내는 `move_map`을 서, 남, 동, 북 버전으로 미리 만들어두면 효율적이다.
