# 19236번. 청소년 상어

## 문제
https://www.acmicpc.net/problem/19236

---

## Key Points

1. 시뮬레이션 + 백트래킹으로 획득 가능한 최대 점수를 탐색하는 문제이다.

2. 시뮬레이션 구현은 아래와 같이 나누어 생각하면 좋다.

    - 물고기 움직이기
        - 물고기를 번호 순으로 움직이기 위해, `pos_map` 배열 사용 (매번 찾을 필요 없음)

    - 상어 움직이기 + 먹기
        - 백트래킹으로 가능한 경우 모두 탐색 (4x4 배열이므로, 한 번에 최대 3개의 선택지가 있음)

3. 백트래킹 할 때 `move_fish()`로 인한 변화와 `eat()`로 인한 물고기 위치 변화를 되돌려 줘야 한다.

    - 1차원 배열의 경우 shallow copy를, 2차원 배열의 경우 list comprehension을 활용한 shallow copy를 사용했다.

    - `copy.deepcopy()`를 사용해도 좋지만, PS 문제이기 때문에 속도가 더 빠른 shallow copy를 사용했다.

4. `pos_map`을 처음에 한 번 만들고 계속 사용하는 방법과, `move_fish()` 함수에서 매번 만드는 방법이 있다.

    - 전자는 시간상 효율적이지만, `pos_map`을 계속 들고 다니고 처리해 줘야하기 때문에 구현이 복잡해지고 가독성이 떨어진다: [[solve.py](./solve.py)]

    - 후자는 `pos_map`을 사용할 때만 만들기 때문에 구현이 간단하다. 전자에 비해 비효율적이지만, 4x4 보드이기 때문에 고려해볼만 하다: [[solve_simple.py](./solve_simple.py)]
