# 20061번. 모노미노도미노 2

## 문제
https://www.acmicpc.net/problem/20061

---

## Key Points

0. 지문이 매우 복잡하고 어려워보이지만, 사실 [19235번. 모노미노도미노](../19235/) 문제의 쉬운 버전이다.

1. 문제를 차근차근 이해해보면 아래와 같이 요구사항을 정리할 수 있다.

    - 블록 놓기
        - 중력을 적용하여 떨어트리기

    - 완성된 행이 있는지 체크하여 제거하기
        - 제거 후, 위에 있던 행들을 한 칸씩 밀기
        - 완성된 행이 없을 때 까지 반복

    - 연한 행에 블록이 있다면 끝 행 제거하기

2. 파란색 보드는 초록색 보드와 (x, y) 좌표를 기준으로 대칭이다. 즉, 하나의 보드만 생각하고 구현하면 된다.

3. 블록 놓기 구현의 경우, 일단 블록을 (0, col)에 놓고 중력을 적용하는 방식으로 구현한다. (`add_block()`)

4. 완성된 행 처리의 경우, 행 제거만 해주고 한 칸 밀어주면 되어서 고려할 사항이 적다. (`delete_row()`)

    - `del`을 사용하여 제거 후, 0번째 행을 [0, 0, 0, 0]으로 채워주면 된다.

    - [19235번. 모노미노도미노](../19235/) 문제는 중력을 다시 적용해주어야 한다.
