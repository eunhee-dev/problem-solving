# 14502번. 연구소

## 문제
https://www.acmicpc.net/problem/14502

---

## Key Points

1. 이 문제는 아래와 같은 단계로 나누어 생각할 수 있다.

    - 빈 칸과 바이러스 위치 구하기

    - 빈 칸에서 벽 3개 세우는 조합 구하기

    - 각각의 조합에 대해 벽을 세운 후 바이러스를 전파시키고 안전영역 구하기

2. 벽 3개 조합 구하는 것은 `itertools.combinations`로 쉽게 구현 가능하다.

3. 바이러스를 전파시키고 안전영역을 구하는 것은 bfs로 쉽게 구현 가능하다. [[풀이 - solve.py](./solve.py)]

    - `temp_board = [row[:] for row in board]`로 각 시뮬레이션 별 `temp_board`를 만들어 준다.

4. board를 직접 변경하고, bfs가 끝나면 변경된 좌표를 되돌리는 방식(undo)으로도 구현할 수 있다. [[풀이 - solve_undo.py](./solve_undo.py)]

    - undo 방식은 각각의 조합마다 board를 새로 만들어줄 필요가 없어서 효율성 측면에서 더 좋을 수 있다.

    - 이 문제는 N이 매우 작아 시간복잡도가 여유롭지만, 더 타이트한 문제에서 undo를 적용해 볼만 하다.
        - 참고로, 이 문제에서는 undo 버전의 실행 시간이 200ms 정도 더 길게 채점되었다.
        - N이 작아서 보드를 그냥 새로 만들어 버리는게, 바꾼 좌표를 추적하는 것보다 싸게 먹힌 것으로 보임.

    - 코드의 효율성과 간결성의 trade-off를 생각해 볼 수 있는 문제
