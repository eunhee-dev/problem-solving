# 13460번. 구슬 탈출 2

## 문제
https://www.acmicpc.net/problem/13460

---

## Key Points

1. 굉장히 복잡한 문제이지만, 차근 차근 아래 단계대로 생각해보면 풀 수 있다.

    - 빨간 구슬, 파란 구슬, 구멍 위치 찾기 => 쉬움 (`find_pos()`)

    - 4 방향으로 `board` 기울이기
        - `board`를 기울였을 때 구슬들의 위치 구하기
            - 구슬들이 동시에 움직여야 함

        - 각 구슬이 구멍에 들어갔는지 확인하기

        - 해당 버전을 기준으로 `board` 다시 기울여보기

2. 이전에 기울인 방향이 다음 기울이기에 영향을 미친다.

    - bfs를 통해 queue에 다음 구슬들의 위치를 시행횟수 순으로 넣어주면, 빨간 구슬이 구멍에 들어가는 순간이 최소 시행횟수이다.

    - 각 구슬이 벽에 닿기 직전 또는 구슬이 구멍에 들어간 순간 멈춰야 한다.

    - 구슬을 동시에 움직이는 것을 구현해야 하는데, 지금 코드는 구슬을 독립적으로 이동시켜 겹칠 수 있다.
        - 각각 이동 거리를 계산하여, 이동거리가 더 긴 구슬은 움직인 방향 반대로 한 칸 후진시켜야한다.

    - visited 배열을 set으로 구현하면 4차원 배열을 만들 필요가 없다.
