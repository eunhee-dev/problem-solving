# 17822번. 원판 돌리기

## 문제
https://www.acmicpc.net/problem/17822

---

## Key Points

1. 이 문제는 아래와 같이 나누어 생각하면 좋다.
    
    - 원판 돌리기 => `deque.rotate()` 활용

    - 인접하면서 같은 수 찾기 => `bfs` 활용
        - 지운 수를 0으로 표기 (원판의 수가 0이 될 수 없기 때문에 가능)

    - 인접하면서 같은 수가 없으면, 평균을 구하여 원판에 반영
        - 모두 지워진 경우를 예외 처리 해주어야함 (Division by zero)

2. 원판이기 때문에, 열에 대해서는 원형(toroidal)으로 연결된 bfs를 사용해야 함
    - `ny = (y + dy) % m` 으로 계산
