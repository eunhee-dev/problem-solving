# 17825번. 주사위 윷놀이

## 문제
https://www.acmicpc.net/problem/17825

---

## Key Points

1. 백트래킹 문제이지만, 사실 윷놀이 판을 구현하는 것이 메인인 문제이다.

2. 인덱스 0: 출발, 32: 도착인 `board`를 구성하고, `blue`를 통해 파란 화살표 이동을 구현, `nxt`를 통해 인덱스의 간극을 없애준다.

    - 예를들어 점수 40 칸의 index는 20인데, 도착 칸은 32이다. 이 간극을 메워주기 위해 `nxt` 배열을 만들고 `nxt[20] = 32`와 같이 정의해준다.

    - `blue`와 `nxt`를 잘 만들었다면, `blue`와 `nxt`에 따라 말을 1칸씩 이동해보면 된다.
