# 23291번. 어항 정리

## 문제
https://www.acmicpc.net/problem/23291

---

## Key Points

1. 다음과 같이 나누어 구현하면 좋다.

    - 물고기 채우기

    - 어항 옮기기
        - 제일 왼쪽 어항 옮기기
        - 2개 이상 쌓여있는 어항 옮기기

    - 어항 조정하고 평탄화하기

    - 공중 부양하기

2. 자료구조를 잘 선택하는 것이 중요한 문제이다.

    - 나는 앞에서 넣고 뺄게 많은 것 같아 `deque[deque[int]]`를 사용하였다.

    - 넣고 빼는 것을 문제 조건대로 구현하면 생각보다 헷갈리는 지점이 적다.

3. 인접한 어항 간의 차이를 계산할 때는 오른쪽과 아래 방향만 계산하면 중복을 피할 수 있다.

    - [23289번. 온풍기 안녕!](../23289/) 문제와 동일한 원리이다.

    - 동시에 반영되어야 하므로, 따로 diff 배열을 만들어서 기록한 후 한 번에 처리하는 것이 좋다.
