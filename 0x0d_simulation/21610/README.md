# 21610번. 마법사 상어와 비바라기

## 문제
https://www.acmicpc.net/problem/21610

---

## Key Points

1. 문제에서 구현하라는대로 구현하면 되는 문제이다.

    - 좌표는 0-based를 사용했고, `DIRECTION`의 경우 편의상 앞에 `(0, 0)` dummy를 추가해 1-based를 사용하였다.

    - 구름 이동은 경계를 넘어가도 되기 때문에 `% n`을 해주어야 한다.

    - 구름에서 비를 다 내리게 한 다음, 물복사버그 마법을 계산해야 한다.

    - 구름이 사라진 자리에는 구름이 생기면 안되기 때문에, 이전 구름의 위치를 `prev_clouds`로 기억하고 있는다.
        - set으로 바꿔주면 구름이 있었는지 효율적으로 알 수 있다.
