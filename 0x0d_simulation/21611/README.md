# 21611번. 마법사 상어와 블리자드

## 문제
https://www.acmicpc.net/problem/21611

---

## Key Points

1. 구현이 복잡한 문제이기 때문에, 잘 나누어 생각해야 한다.

    - 블리자드 구현 (2차원 배열 기준)

    - 폭발 구현 (나선 순으로 연속한 구슬 확인)

    - 변화 구현 (나선 순으로 연속한 구슬 확인)

    - 이 문제와 같이 구현량이 많은 문제는 실수를 하기 쉽기 때문에, 개인적으로 최적화보다는 가독성을 중점으로 코딩하는 것이 더 좋은 것 같다.
        - 비슷한 로직 또는 복잡한 로직은 함수화
        - (불필요해 보이더라도) 일단 문제에 나와있는대로 구현하기

2. 폭발/변화를 구현하기 위해 나선 순으로 연속한 구슬을 체크해야하기 때문에, 1차원 배열을 사용하면 좋다.

    - 상어를 시작으로 나선 순으로 1차원 배열로 변환 (`board_1d`)

    - 다만, 블리자드 구현은 2차원 배열을 사용하기 때문에, 2차원 배열 좌표를 1차원 배열의 인덱스로 변환하는 맵을 가지고 있으면 편할 것 같다. (`board_2d_to_1d`)
        - 블리자드로 파괴되는 좌표는 2차원으로 계산하고 맵을 통해 1차원 배열의 인덱스를 구하면 됨

3. 연속한 구슬을 확인하는 것은 `grouping()` 함수를 만들어 폭발/변화에서 공동으로 사용 가능하다.

    - 폭발에서는 4개 이상 연속한 경우를 사용

    - 변화에서는 전부 사용

    - 마지막에 연속된 구슬로 끝나는 경우의 처리를 잘 해줘야 한다. (빠트리기 쉬움)
        - python의 for-else문을 사용하면 간결하게 처리 가능하다. (break 없이 끝난 경우 후처리 가능)

4. 변화에서 경계 부분 체크를 잘 해주어야 한다.

    - 구슬이 칸의 수를 넘어가는 경우에 대해 문제 조건에 따라 잘 처리해줘야 한다. (마지막 칸에 있는 구슬은 변화하지 못하고 사라져야 한다.)
