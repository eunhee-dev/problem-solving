# 14890번. 경사로

## 문제
https://www.acmicpc.net/problem/14890

---

## Key Points

1. 문제를 잘 이해해야 풀 수 있다. 지문과 예제 입출력을 보면, 각 line별로 독립적인 것을 알 수 있다.

2. 각 line 별로 경사로를 놓아 지나갈 수 있는지 체크해주면 된다.

    - 문제 조건에 맞게 경사로 놓기

    - `visited` 배열을 활용해 이미 경사로를 놓은 칸인지 체크

    - 문제 풀다가 까먹을 수 있는데, 높이가 1 차이 날때만 경사로를 놓을 수 있다.
        - 높이가 2 차이나면 바로 False를 return 해주면 된다.
        - 놓치기 쉬운 부분
