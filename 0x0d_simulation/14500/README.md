# 14500번. 테트로미노

## 문제
https://www.acmicpc.net/problem/14500

---

## Key Points

1. 이 문제는 아래와 같이 두 단계로 생각하면 좋다.

    - 시작점 선정하기

    - 각 시작점에서 깊이 4의 백트래킹으로 합 구하기

2. 깊이 4의 백트래킹으로 'ㅜ' 모양을 제외하고 모든 모양을 체크 가능하다.

    - 'ㅜ' 모양의 경우 칸 3개 탐색이 끝난 시점에서 3번째 칸 (nx, ny)가 아닌 2번째 칸 (x, y)를 다음 탐색점으로 넣어주면 된다.

3. 시작점을 왼쪽에서 오른쪽으로 넣어주기 때문에, 왼쪽 탐색 (0, -1) 없이 3방향 탐색으로도 완전 탐색이 가능하다.

    - 중복되는 탐색을 줄일 수 있어 약간 더 효율적이다.

4. n * m 시작점에 대해 깊이 4의 백트래킹을 진행하므로 O(n * m)에 풀이 가능하다.

5. 테트로미노를 미리 만들어 놓은 후, n * m 시작점에 다 넣어보는 방법도 있다. 중복되는 탐색이 없언 더 효율적일 것으로 생각된다.
