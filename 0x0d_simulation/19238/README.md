# 19238번. 스타트 택시

## 문제
https://www.acmicpc.net/problem/19238

---

## Key Points

1. bfs를 활용하여 문제에 적혀있는대로 구현하면 되는 문제이다.

    - bfs로 다음에 태울 승객과 거리를 찾는다. (`pick_up_bfs`)
        - 같은 거리의 승객이 여러명 있으면 우선순위에 맞게 정렬하여 고른다.
        - 승객이 있는데 막혀있어서 태울 수 없거나, 연료가 부족하면 return -1

    - bfs로 도착지까지 거리를 구한다. (`drop_off_bfs`)
        - 마찬가지로 도착지에 가는 길이 막혀있거나, 연료가 부족하면 return -1
