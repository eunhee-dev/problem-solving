# 2573번. 빙산

## 문제
https://www.acmicpc.net/problem/2573

---

## Key Points

1. 다음과 같은 과정으로 문제를 나눠볼 수 있음
    - 빙산 녹이기 (시뮬레이션)
    - 빙산 덩어리 개수 카운트 (bfs)

2. 빙산 녹이기 (시뮬레이션)
   
    - 빙산을 녹일 때, `board`에 실시간으로 반영하면 안됨
        - 빙산의 높이는 동시에 줄어들어야함
        - 어떤 칸 A가 녹아서 0이 되었다고 해서, 그 해에 바로 옆 칸 B가 녹는 양에 영향을 주면 안됨

    - `melting_board`를 이용해서 각 칸에 대해 줄어들 빙산의 높이를 미리 계산

    - 모든 칸에 대해 계산이 끝난 후에 `melting_board`의 값을 통해 실제로 빙산(`board`)을 녹여야함

    - 빙산에 대해서만 수행하고 바닷물인(`board[x][y] == 0`) 경우 스킵하면 약간의 효율성을 얻을 수 있음

3. 빙산 덩어리 개수 카운트 (bfs)
    - 기본적인 bfs를 통한 영역 개수 세는 문제와 동일
