# 16920번. 확장 게임

## 문제
https://www.acmicpc.net/problem/16920

---

## Key Points

1. 하나의 점에 대해 끝까지 탐색하는게 아니라, 중간에 끊고 다음 플레이어에게 턴을 넘겨줘야 한다. (이로 인해 `board`가 바뀔 수 있다.)

2. 이를 구현하기 위해 플레이어 별로 `queue`를 저장하면 된다. (여기서 `queue`는 탐색을 마친 시점에서 자기 영역의 테두리를 의미한다.)

3. 자기 턴에 움직일 수 있는 만큼 움직여서 `board`의 `.`을 내 숫자로 바꾼다. 그러면 `queue`에는 자기 영역의 테두리가 남게 된다. 이 `queue`를 저장해놓고 턴을 넘긴다.

4. 다음에 자기 턴이 넘어왔을 때 `queue`에 저장된 값(테두리 == 출발점)을 하나씩 꺼내어 3번 과정을 수행한다.

5. 모든 플레이어가 더 이상 확장할 수 없을 때까지 while문을 돌리면서 진행하면 된다.

6. 구현 관련 소소한 내용

    - 플레이어가 1부터 시작하기도 하고, `board`가 문자로 구성되어 있어서 그냥 딕셔너리를 사용하여 구현하였다.

    - `board`의 수정 소요가 있기 때문에, 입력이 공백 없이 들어옴에도 불구하고 `list[str]`과 같이 문자열로 받는 대신 `list[list[str]]`로 문자를 저장하였다.
        - 문자열은 immutable 하기 때문에 문자열로 받을 경우 수정이 어렵다.
