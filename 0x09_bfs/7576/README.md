# 7576번. 토마토

## 문제
https://www.acmicpc.net/problem/7576

---

## Key Points

1. 여러 개의 시작점에서 전파하는 문제이다. (멀티소스 bfs 문제)

    - 시작점 마다 bfs를 돌리게 되면, O(N * M * K), K = 처음에 익은 토마토 수

    - 큐에 시작점을 모두 넣고 bfs 돌리면, 각 빈 칸까지의 가장 가까운 익은 토마토로부터 전파되는 최소 거리를 한 번에 계산할 수 있음

2. dist 에서 최대 거리를 찾으면 된다.

    - 만약 dist에 -1이 있는데 토마토가 안들어 있는 칸이 아니면 다 못 채운 것, return -1
