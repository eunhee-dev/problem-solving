# 2504번. 괄호의 값

## 문제
https://www.acmicpc.net/problem/2504

---

## Key Points

1. 스택에 괄호 뿐만 아니라 값도 넣어보자.
    - 문제 설명과 같이 (2) 이렇게 그대로 stack에 넣기 (닫는 괄호 빼고)
    - stack = ["(", 2]
  
2. 닫는 괄호가 나오면,
    - 앞에 쌓여있는 정수들을 모두 빼서 더해주고, (to_be_sum)
    - stack의 top이 맞는 쌍인지 확인한 후 pop하고, ")" => "("
    - 앞에 정수들이 있었으면 해당하는 값 (2 또는 3)이랑 곱해준다. to_be_sum * val
    - 없으면 단일 괄호("()", "[]")로 2 또는 3을 넣는다. to_be_sum

3. 마지막으로 stack에 아직 괄호(str)가 포함되어 있는지 체크해준다.
    - "(()" 케이스와 같이 다 안닫힌 경우 방지

4. 아직 안 더해진 stack 안에 있는 숫자들을 모두 더해 return 한다.
