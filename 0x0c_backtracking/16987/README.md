# 16987번. 계란으로 계란치기

## 문제
https://www.acmicpc.net/problem/16987

---

## Key Points

1. N이 8 이하여셔 백트래킹을 통해 완전 탐색을 할 수 있다. (계란 들어서 이 계란 깨보고, 저 계란 깨보고). (가지치기를 고려 안하면 최대 7! 정도의) 모든 경우의 수를 시뮬레이션 해본다고 생각하면 된다.

2. 문제 요구대로 구현해주어야 한다. 깨진 계란을 집거나, 더 이상 깰 계란이 없는 경우를 잘 처리해주어야 한다.

    - 깨진 계란을 잡은 경우 (`is_broken[curr_idx] = True`): 다음 계란 잡기 (`backtrack(curr_idx + 1)`)
  
    - 더 이상 깰 계란이 없는 경우 (`target_remained = False`): 종료하기 (`backtrack(n)`)
