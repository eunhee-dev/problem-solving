# 1463번. 1로 만들기

## 문제
https://www.acmicpc.net/problem/1463

---

## Key Points

1. dp를 사용하여 i = 2 ~ n까지 `dp[i]`를 채운다.

    - dp[i]: i를 1로 만들기 위해 필요한 연산 사용 횟수의 최소값

    - dp는 점화식 정의가 매우 중요하다.

    - dp는 1-based로 푸는 것이 일반적인 것 같다. (직관적이며, 실수를 줄일 수 있는 장점?)

2. bfs를 통해서도 풀이 가능하다.

    - bfs는 1에서 출발하여 x3, x2, +1로 탐색하면서 가장 빨리 n에 도달하는 거리를 찾으면 된다.

    - bfs가 대부분의 상황에서 효율이 더 좋지만, dp는 코드가 더 간결해진다.
