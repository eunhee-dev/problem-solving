# 14002번. 가장 긴 증가하는 부분 수열 4

## 문제
https://www.acmicpc.net/problem/14002

---

## Key Points

1. [11053번. 가장 긴 증가하는 부분 수열](../11053번.%20가장%20긴%20증가하는%20부분%20수열/) 문제에서 경로 역추적/복원을 추가해주면 되는 문제이다.

    - 경로 역추적/복원은 [12852번. 1로 만들기 2](../12852번.%201로%20만들기%202/) 문제 풀이처럼 하면된다.

    - prev[i]: 수열에서 i 직전에 있는 수의 인덱스를 기록

    - 더 긴 수열이 있어 dp 테이블이 갱신될 때, prev 배열도 같이 갱신해준다.

    - 수열에서 가장 큰 수에서 부터 prev 배열을 역추적하여 경로를 복원하면 된다.
