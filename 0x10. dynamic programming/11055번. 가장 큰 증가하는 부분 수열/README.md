# 11055번. 가장 큰 증가하는 부분 수열

## 문제
https://www.acmicpc.net/problem/11055

---

## Key Points

1. 1차원 dp의 기초적인 O(n) 구조가 아니라 생소할 수 있는데, dp로 풀이 가능하다.

    - dp[i]를 계산하기 위해, 이전 상태(dp[0] ~ dp[i - 1])를 모두 확인해야하는 문제

2. dp[i]: i번째 숫자로 끝나는 증가하는 부분 수열의 합의 최대 값

    - i번째 숫자로 끝나도록 정의한 이유는, 뒤에 붙일 수 있는지 없는지 판단하기 위해 마지막 숫자가 무엇으로 끝나는지에 대한 정보가 필요하기 때문이다.

    - a[i]로 끝나는 가장 큰 합의 수열을 만들려면, a[i] 이전의 수열 또한 최적의 수열이어야 한다. (dp 구조)

    - dp[i]를 a[i]로 초기화 해두고, 0 <= j < i 인 j에 대해 a[i] > a[j] 인 경우 `max(dp[i], dp[j] + a[i])`를 찾으면 된다.
        - 1-based로 바꿨을 때 얻는 이점이 없어서, 0-based를 유지하였다.

    - 마지막 수까지 다 계산했다면, dp 배열의 최대 값이 정답이 된다. (마지막 수를 포함한다고 가장 큰 값이 아니기 때문에 `dp[n - 1]`은 정답이 아님)

    - 조금 비효율적으로 느껴질 수 있는데, n이 1000 이하이기 때문에 O(n^2)의 풀이도 통과 가능하다. (더 효율적인 풀이도 존재하지만, 현재 강의 범위를 넘어선다.)
