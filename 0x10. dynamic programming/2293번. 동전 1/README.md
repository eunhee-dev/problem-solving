# 2293번. 동전 1

## 문제
https://www.acmicpc.net/problem/2293

---

## Key Points

1. dp[i]: 현재까지 사용한 동전들로 i원을 만드는 경우의 수 

    - (각 코인에 대해) dp[i] = (현재 coin를 사용하지 않고 i를 만드는 경우의 수) + (현재 coin를 사용하여 i를 만드는 경우의 수)

    - 예를 들어, 1원 코인을 이미 계산해놓은 시점에서 `2원을 만드는 경우의 수 = 2원을 사용하지 않고 2원을 만드는 경우의 수(1 + 1) + 2원을 사용하여 2원을 만드는 경우의 수(0 + 2)`이다.

    - 코드로 표현하면 점화식은 아래 형태가 된다.
  
        ```python
        for coin in coins:
            for i in range(coin, k + 1):
                dp[i] = dp[i] + dp[i - coin]
        ```

        - 이런 형태의 점화식을 세우는 것이 조금 낯설 수 있다.

        - dp[0] = 1로 두어야 점화식을 세우기 편하다. (의미적으로도 0원을 만드는 경우의 수는 아무 동전도 사용하지 않는 1가지이므로 맞음)
            - 그렇지 않으면, `dp[coin]`을 예외처리 해주고 i를 coin + 1부터 시작해야 한다.

        - coins는 정렬하지 않아도 상관 없다.
