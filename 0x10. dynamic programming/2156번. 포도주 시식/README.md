# 2156번. 포도주 시식

## 문제
https://www.acmicpc.net/problem/2156

## Key Points

1. [2579번. 계단 오르기](../2579번.%20계단%20오르기/)와 비슷한 문제이다.

    - 계단 오르기는 마지막 계단을 무조건 밟아야 하지만, 이 문제는 그렇지 않다.

2. 2차원 dp를 사용하여 연속으로 밟은 상태를 표시하면 된다. [[풀이 - solve.py](./solve.py)]

    - dp[i][j]: i번째 포도주 위치까지 j번 연속해서 포도주를 먹었을 때 최대로 마실 수 있는 포도주의 양 (j = 0 은 i번째 포도주를 먹지 않음)

    - `dp[i][0] = max(dp[i - 1])`: i번째 포도주를 먹지 않으면, `max(dp[i-1])`(i-1번째 포도주 위치까지 왔을 때 최대로 마실 수 포도주의 양임)

    - `dp[i][1] = max(dp[i - 2]) + wine[i]`: i-1번째 포도주를 마시지 않고 i번째 포도주를 마신다면, `max(dp[i - 2])`에 `wine[i]`를 더한 값임

    - `dp[i][2] = dp[i - 1][1] + wine[i]`: i번째 포도주가 2번 연속해서 먹은 포도주라면, `dp[i - 1][1]`(i-2번째 포도주를 마시지 않고 i-1번째 포도주를 마신 경우)에 `wine[i]`를 더한 값임

3. 위의 케이스들을 보면 1차원 dp로도 표시할 수 있다. [[풀이 - solve.py](./solve_1d.py)]

    - dp[i]: i번째 포도주 위치까지 왔을 때 최대로 마실 수 있는 포도주의 양

    - `dp[i] = max(dp[i - 1], dp[i - 2] + wine[i], dp[i - 3] + wine[i - 1] + wine[i])`
        - 각각의 요소가 2차원 dp에서 `dp[i][0]`, `dp[i][1]`, `dp[i][2]` 임
