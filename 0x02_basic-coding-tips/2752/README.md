## Key Points

1. `max()`/`min()` 함수로 3개의 수의 최대값/최소값을 구하고, 중간값은 `전체 합 - 최대값 - 최소값`으로 구할 수 있다. [[풀이 - solve.py](./solve.py)]

    - 정렬을 사용하지 않은 풀이 아이디어
    - 정수가 3개로 고정되어 있고 최댓값/최솟값 자체를 바로 구할 수 있기 때문에 사용할 수 있음


2. python 기본 라이브러리 함수인 `sorted()`를 사용하면 간결하게 정렬할 수 있다. [[풀이 - solve_sort.py](./solve_sort.py)
    - 원본 리스트 자체를 정렬하므로`list.sort()`가 더 효율적 
    - 원본 리스트 객체를 **유지**해야 할 필요가 있을 때에는 `sorted()` 함수를 사용할 것
