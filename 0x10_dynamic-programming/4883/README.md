# 4883번. 삼각 그래프

## 문제
https://www.acmicpc.net/problem/4883

---

## Key Points

1. 각 노드에 도달 할 수 있는 최소 비용을 dp 배열에 저장하면 된다.

    - 본 문제는 0-based를 기준으로 삼았다.

    - (i, 0)은 (i - 1, 0), (i - 1, 1) 두 곳에서 올 수 있다.
        - `dp[i][0] = min(dp[i - 1][0], dp[i - 1][1]) + board[i][0]`

    - (i, 1)은 (i - 1, 0), (i - 1, 1), (i - 1, 2), (i, 0) 네 곳에서 올 수 있다.
        - `dp[i][1] = min(dp[i - 1][0], dp[i - 1][1], dp[i - 1][2], dp[i][0]) + board[i][1]`

    - (i, 2)는 (i - 1, 1), (i - 1, 2), (i, 1) 세 곳에서 올 수 있다.
        - `dp[i][2] = min(dp[i - 1][1], dp[i - 1][2], dp[i][1]) + board[i][2]`

    - 같은 행에서 j열은 j - 1열에서 올 수 있기 때문에 0 -> 1 -> 2 열 순서대로 계산해야 한다.

2. (0, 0)은 절대 갈 수 없는 노드이기 때문에, 간결한 예외처리를 위해 `dp[0][0] = sys.maxsize`로 초기화시켰다. 

3. 본 문제도 피보나치 수열과 마찬가지로 dp[i]를 계산하는데 dp[i - 1]과 dp[i]만 필요하기 때문에 굳이 dp 배열에 모두 저장할 필요는 없다.

    - 다만, 개념 이해를 위해 dp 배열을 만들어 풀이하였다.

    - 실제로 구현할 때는, dp[i - 1][0 ~ 3], dp[i][0 ~ 3] 총 6개 변수만 기억하고 있으면 된다.
