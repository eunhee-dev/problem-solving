# 1904번. 01타일

## 문제
https://www.acmicpc.net/problem/1904

---

## Key Points

1. 1과 00 타일 2개로 만들 수 있는 길이가 N인 2진수 개수를 구하면 되는 문제이다.

    - dp[i]: 만들 수 있는 길이가 i인 2진수 개수

    - 마지막 타일이 '1'인 경우: dp[i - 1], 마지막 타일이 '00'인 경우: dp[i - 2]

    - 점화식으로 표현하면 `dp[i] = dp[i - 1] + dp[i - 2]`

    - 결과적으로 피보나치 수열과 점화식이 동일하다.
