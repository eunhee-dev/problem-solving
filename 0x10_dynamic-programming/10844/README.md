# 10844번. 쉬운 계단 수

## 문제
https://www.acmicpc.net/problem/10844

---

## Key Points

1. 전에 어떤 숫자로 끝나는지를 기록하기 위해 2차원 dp를 사용하면 된다.

2. dp[i][j]: j로 끝나는 길이 i의 계단 수 개수

    - (1 <= j <= 8) dp[i - 1][j] = dp[i - 1][j - 1] + dp[i - 1][j + 1]

    - j가 0이거나 9인 경우 각각 dp[i - 1][1], dp[i - 1][8]과 같다.

    - 1000000000으로 나눈 나머지를 구하는 것이니, 덧셈이 일어나는 곳에 대해 나머지를 저장해두면 된다.
        - python의 경우 int가 무한에 가까워 integer overflow 걱정은 없으나, 계산 효율성을 위함
