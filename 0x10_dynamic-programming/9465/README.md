# 9465번. 스티커

## 문제
https://www.acmicpc.net/problem/9465

---

## Key Points

1. 점화식을 세우면 dp로 쉽게 해결 가능한 문제이다.

    - dp[j][i]: (j, i)에 스티커를 붙였을 때 얻을 수 있는 최대 점수 (`0 <= j <= 1`)
        - 구현의 편리함을 위해 행의 경우에는 0-based를 사용하였다.

    - (0, i)에 스티커를 붙였을 때 최대 점수가 되기 위해서는, (0, i - 1)에 스티커가 붙어있는 경우를 제외하고 고려하면 된다. 즉, (1, i - 1)에 스티커가 붙어있거나, i - 2번 열에 스티커가 붙어있는 경우에만 가능하다.
        - `max_dp_i_2 = max(dp[0][i - 2], dp[1][i - 2])`
        - `dp[0][i] = max(dp[1][i - 1], max_dp_i_2) + sticker[0][i]`
        - `dp[1][i] = max(dp[0][i - 1], max_dp_i_2) + sticker[1][i]`
    
    - 자세히 생각해보면, (0, i)에 스티커를 붙였을 때 최대 점수가 되기 위해서는, i - 2번 열의 경우 (1, i - 2)만 고려해주면 된다.
        - 왜냐하면, (0, i - 2)에 스티커를 붙인 경우, (1, i - 1) -> (0, i) 순으로 스티커를 붙이는 것이 더 좋기 때문이다.
        - 즉, (0, i - 2)에 스티커를 붙인 경우는 dp[1][i - 1]에 포함되어 있다.
        - 따라서 최종 점화식은 아래와 같다.
            - `dp[0][i] = max(dp[1][i - 1], dp[1][i - 2]) + sticker[0][i]`
            - `dp[1][i] = max(dp[0][i - 1], dp[0][i - 2]) + sticker[1][i]`
