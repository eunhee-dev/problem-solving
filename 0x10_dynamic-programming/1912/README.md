# 1912번. 연속합

## 문제
https://www.acmicpc.net/problem/1912

---

## Key Points

1. 연속합이기 때문에, dp를 활용하여 각 위치에 대해 최대 연속합을 기록해두면 된다.

    - dp[i]: i번째 수를 끝으로 포함시켰을 때 최대 연속합

    - dp[i - 1]이 양수이면 더하고 아니면 더하지 않는 것이 최대 연속합이 된다.

    - `dp[i] = dp[i - 1] + seq[i] if dp[i - 1] > 0 else seq[i]`

2. 완성한 dp 배열에서 최대 값을 뽑으면 그 값이 가능한 최대 연속합이 된다.
