# 14501번. 퇴사

## 문제
https://www.acmicpc.net/problem/14501

---

## Key Points

1. n이 15 이하이기 때문에 매우 여유롭다. 백트래킹이 먼저 떠오르지만, dp로도 풀이 가능하다.

2. dp[i]: i일에 끝나는 상담을 했을 때 얻을 수 있는 최대 수익

    - 상담이 끝나야 새로운 상담을 시작할 수 있기 때문에, 기준을 끝나는 날을 기준으로 정하면 편하다.

    - `i + t - 1`일에 상담을 끝냈을 때 얻을 수 있는 최대 수익은, `i일 이전에 얻을 수 있는 수익 중 최대 값 + p_i` 중 최대값 이다.

    - 수식으로 표현하면, `dp[i + t - 1] = max(dp[i + t - 1], max(dp[:i]) + p)`

    - 상담을 언제 끝내는게 최대 이익일지 모르기 때문에, `max(dp)`가 원하는 답이 된다.

3. 이 방식은 1 <= i <= n 인 i에 대해 `max(dp[:i])`를 매번 계산해야하기 때문에 O(n^2)의 시간복잡도를 가진다.

    - [15486번. 퇴사 2](../15486/) 문제와 같이 n이 큰 경우에 불가능한 풀이이다.

    - `max(dp[:i])` 부분을 효율적으로 처리하면 O(n)에 풀이 가능하다.

    - 더 효율적인 풀이는 [15486번. 퇴사 2](../15486/) 문제에서 설명한다.
