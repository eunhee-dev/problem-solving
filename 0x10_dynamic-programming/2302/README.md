# 2302번. 극장 좌석

## 문제
https://www.acmicpc.net/problem/2302

---

## Key Points

1. dp 배열을 어떻게 정의하고 활용하는지에 따라 2가지 방법으로 풀이할 수 있다.

    - dp 배열을 통해 바로 문제를 해결하는 방법

    - dp 배열을 통해 부분 문제를 해결하는 방법

2. dp 배열을 통해 바로 문제를 해결하는 방법 [[풀이-solve.py](./solve.py)]

    - dp[i]: i번째 좌석까지 주어진 조건을 만족하면서 사람들이 좌석에 앉을 수 있는 방법의 가짓수

    - i번 또는 i-1번이 지정석인 경우에는 i번 자리가 고정되기 때문에 `dp[i] = dp[i - 1]` 이다.
        - i-1번이 지정석인 경우는 사실 `dp[i - 2]`인데, 이전 연산에서 `dp[i - 1] = dp[i - 2]`를 했기 때문에 `dp[i - 1]` 이어도 무방하다.

    - 그렇지 않은 경우에는 i번과 i-1번을 바꿀 수 있기 때문에 `dp[i] = dp[i - 1] + dp[i - 2]` 이다.
        - i번 자리에 i가 앉을 경우 `dp[i - 1]` + i-1번 자리에 i가 앉고 i번 자리에 i-1이 앉을 경우 `dp[i - 2]`

    - dp[0]은 존재하지 않는 값이지만, `dp[0] = 1`로 하면 dp[2]를 점화식으로 처리 가능하기 때문에, `dp[0] = 1`로 두었다.
        - 그렇지 않으면, `dp[2] = 1 if (i in vip) or (i - 1 in vip) else 2`와 같이 처리해야 한다.
        - 의미상으로도 빈 좌석의 경우의 수는 1가지이긴 하다.
    

3. dp 배열을 통해 부분 문제를 해결하는 방법 [[풀이-solve2.py](./solve2.py)]

    - 지정석을 고려하지 않고 가지수를 구하면 피보나치 수열이 된다. `dp[i] = dp[i - 1] + dp[i - 2]`

    - 지정석은 고정되어 있기 때문에, 지정석을 기준으로 나누어 각 가짓수를 곱하면 된다.
        - 예를 들어 n = 9에서 4, 7번이 지정석이라면 `1 ~ 3번 배치 가짓수` * `5 ~ 6번 배치 가짓수` * `8 ~ 9번 배치 가짓수`로 구할 수 있다.
        - `1 ~ 3번 배치 가짓수 = dp[3]`, `5 ~ 6번 배치 가짓수 = dp[2]`, `8 ~ 9번 배치 가짓수= dp[2]` 가 된다.
        - 따라서, dp 배열을 `1 ~ n` 까지 구할 필요 없이, 최악의 경우를 고려해도 `1 ~ n - len(vip)`까지만 구하면 된다.
    
    - dp 배열을 활용하여 각 부분 문제를 해결하고 조합하는 방식이다.
