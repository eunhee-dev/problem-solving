# 11052번. 카드 구매하기

## 문제
https://www.acmicpc.net/problem/11052

---

## Key Points

1. dp로 쉽게 해결 가능한 문제이다.

    - dp[i]: i개의 카드를 갖기 위해 지불해야하는 최대 금액

    - `dp[i] = max(p[j] + dp[i - j])`, (`1 <= j <= i`) 로 표현할 수 있다.
        - 카드팩과 나머지 최적해의 조합
        - j는 1 ~ i까지 모두 확인해야한다.

    - dp 배열을 p로 초기화해버리면 `dp[i] = max(dp[j] + dp[i - j])`, (`1 <= j <= i // 2`) 로 표현할 수 있다. [[풀이 - solve.py](./solve.py)]
        - 두 최적해의 조합, dp[1] ~ dp[i - 1]은 모두 최대 금액을 보장한다.
        - `dp[j] + dp[i - j]`는 `dp[i - j] + dp[j]`와 완전히 같은 계산이므로, 중복을 피해 j를 i의 절반까지만 탐색해도 된다.
