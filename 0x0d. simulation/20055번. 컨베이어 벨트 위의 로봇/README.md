# 20055번. 컨베이어 벨트 위의 로봇

## 문제
https://www.acmicpc.net/problem/20055

---

## Key Points

1. 문제를 잘 읽고 요구사항대로 구현하면 된다.

   - 컨베이어 벨트 회전 => `deque()` 사용
      - `n - 1` 인덱스에 도달하면 로봇 내리기

   - 로봇 움직이기 (가장 먼저 벨트에 올라간 로봇부터 => `n - 2`부터 역순으로)
      - 내구도 감소시키기
      - `n - 1` 인덱스에 도달하면 로봇 내리기

   - `0` 인덱스에 로봇 올리기
      - 내구도 감소시키기

2. 가독성을 고려하면 `Block`을 `list[int]`로 구현하는 것보다 `dataclass`를 사용하면 좋다. [[풀이 - solve_dataclass.py](./solve_dataclass.py)]

   - 남이 보고 이해하기에도 좋고, 문제 요구사항이 복잡해져도 코딩하는데 덜 헷갈린다.

   - `list`에 비해 시간적으로 손해는 발생하지만, 무시 가능한 수준이다.
