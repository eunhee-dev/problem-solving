# 15686번. 치킨 배달

## 문제
https://www.acmicpc.net/problem/15686

---

## Key Points

1. 이 문제의 경우도 아래와 같이 두 단계로 나누어 생각하면 좋다.

    - 치킨 집 M개 조합 뽑기

    - 치킨 집 조합에 따라 도시의 치킨 거리 구하기

2. 위 단계에 따라 구현해주면 된다. [[풀이 - solve.py](./solve.py)]

    - 치킨 집 M개 조합 뽑는 것은 `itertools.combinations`를 사용하면 쉽다.

    - 도시의 치킨 거리 구하는 것도 `abs()`를 통해 맨해튼 거리를 구해주면 된다. (`get_chicken_dist()`)

    - 치킨 집 조합 뽑기 13CM (최대 13C6) * 도시의 치킨 거리 구하기 n * m (100 * 13) 이므로 제한시간 안에 충분히 풀이 가능하다.

3. 구현을 하다보면, 치킨집이랑 집 위치는 고정되어 있는데 왜 매번 거리를 구해야하지? 라는 생각이 들 수 있다. [[풀이 - solve_precomputed.py](./solve_precomputed.py)]

    - 각 치킨집에서 모든 집까지의 거리를 미리 계산 해두고, (`dist_matrix[i][j]` : i번째 치킨집에서 j번째 집까지 치킨 거리)

    - 치킨 집 조합 안에서 각 집에서 치킨집까지 최단거리를 `dist_matrix`를 참조하여 뽑으면 조금 더 효율적이다.
        - `abs()` 호출이 적어져서 그렇다. (n * m 번만 호출, 기존에는 13CM * n * m번 호출)

    - 하지만, 이론적으로 시간복잡도는 기존 풀이와 동일하다. (13CM * m * n) 구현 레벨에서 효율성을 높이는 방식이다.
