# 23290번. 마법사 상어와 복제

## 문제
https://www.acmicpc.net/problem/23290

---

## Key Points

1. 다음과 같이 나누어 생각하면 좋다.

    - 복제 마법 시전

    - 물고기 이동

    - 상어 이동: 전수조사하여 최고의 경로 찾기 (`itertools.product`를 활용하였음)

    - 잡아먹기: 냄새 남기기 포함 (`smell_board`에 기록, 2턴 동안 유지되어야 하는 것에 유의)

    - 냄새 제거

    - 복제 완료

2. 물고기를 개별 관리하는 방법 [[풀이 - solve.py](./solve.py)]

    - 물고기 자료구조를 딕셔너리로 하여, 각각의 물고기에 고유한 id 부여하였다.
        - 물고기를 복제해도 중복되지 않게 구분 가능하다.

    - `board`에는 물고기의 id만을 넣어 상어 이동을 계산하는 것에만 활용

    - 직관적이고 바로 떠올릴 수 있는 풀이이다.

3. `board`에 각 방향별 물고기의 개수만 기록하는 방식 [[풀이 - solve2.py](./solve2.py)]

    - 다른 사람들의 풀이를 보고 생각해보니, 물고기를 개별적으로 생각할 필요 없이 각 방향별 물고기 개수만 알면 되었다.
        - 본 문제는 물고기가 동시에 이동하는 등 물고기 각각이 구분되어질 필요가 없기 때문에 결국 물고기의 위치와 방향만 중요하다.

    - 이렇게 하면, 기존에 물고기 개수만큼 for문을 도는 대신, 4 * 4 * 8의 고정된 값만큼만 for문을 돌아도 되어 훨씬 효율적이다.

    - 다만, 물고기의 순서 등 개별성이 중요한 문제의 경우에는 사용할 수 없는 방식이다.
