# 17472번. 다리 만들기 2

## 문제
https://www.acmicpc.net/problem/17472

---

## Key Points

1. [다리 만들기](../0x09.%20bfs/2146번.%20다리%20만들기) 문제와 같이 bfs를 통해 섬을 표시한다.

2. 각 섬간의 최소 다리 길이를 구한다.

    - i -> j 섬의 최소 다리 길이는 j -> i 섬 최소 다리 길이와 같기 때문에 i < j인 경우만 찾으면 된다.

2. 섬 간의 최소 다리 길이를 구했다면, 모든 섬을 연결하는 최소 다리 길이를 골라야 한다. = 최소 스패닝 트리(MST)를 찾는 문제

    - kruskal 알고리즘을 모르면 풀기 어렵다.
        - MST(최소 비용으로 모든 정점을 연결하는 간선들의 집합)를 찾는 알고리즘
    
    - Union-Find 알고리즘을 사용하여 구현할 수 있다. (사이클이 생기지 않으면 트리에 포함)
