# 17136번. 색종이 붙이기

## 문제
https://www.acmicpc.net/problem/17136

---

## Key Points

1. 붙일 수 있는 색종이를 하나씩 붙여보면서 모든 경우를 시뮬레이션 해봐야 하는 문제이다.

    - 백트래킹을 사용하면 가지치기를 통해 좀 더 효율적인 탐색이 가능하다. (최소 색종이 보다 더 많이 붙이는 경우 등)

    - 값이 1인 좌표들만 보면 조금 더 효율적으로 구현 가능하다.

    - 값이 1인 다음 인덱스를 미리 계산해두면 (재귀를 안에서 검증하는 것 보다) 더 효율적이다.
        - `get_next_idx()` 함수를 구현해서 다음 인덱스를 미리 찾는 것이, 그냥 `backtrack(idx + 1, cnt + 1)`해서 넘긴 후 `if board[x][y] != 1: backtrack(idx + 1, cnt)`를 재귀 안에서 검증하는 것 보다 효율적
