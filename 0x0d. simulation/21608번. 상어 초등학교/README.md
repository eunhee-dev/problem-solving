# 21608번. 상어 초등학교

## 문제
https://www.acmicpc.net/problem/21608

---

## Key Points

1. 다음과 같이 나누어 생각하면 좋다.

    - 1~3번 조건 고려하여 자리 배치하기

    - 점수 구하기

2. 간단하게 모든 빈칸을 순회하여 각 조건에 대해 count 하고 정렬해버리는 방법이 있다. [[풀이 - solve.py](./solve.py)]

    - 각 빈칸에 대해 (1) 인접한 좋아하는 학생 수 (2) 인접한 빈칸 수를 count하고 정렬하여 좌표를 찾는다.

    - n이 20이하 이기 때문에 넉넉하게 통과 가능하다.

3. 좀 더 효율적인 방법으로는, 모든 빈칸이 아닌 좋아하는 학생 주변 빈칸들만 탐색하는 방법이 있다. [[풀이 - solve_efficient.py](./solve_efficient.py)]

    - 좋아하는 학생이 아직 배치 전이거나 근처에 배치할 수 없다면, 모든 빈칸을 확인한다.

    - 좋아하는 학생 주변에 빈칸이 있다면, 모든 빈칸을 확인할 필요없이 그 학생 주변만 확인해보면 된다. (조건 1이 가장 우선순위가 높기 때문에)

    - \+ 굳이 정렬(N^2log(N^2))을 하지 말고, 최적의 좌표(N^2)만 뽑으면 더 효율적이다. (최적의 좌표 외에는 순서를 신경 쓸 필요가 없기 때문에)
