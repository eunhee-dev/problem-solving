# 17140번. 이차원 배열과 연산

## 문제
https://www.acmicpc.net/problem/17140

---

## Key Points

1. 문제의 세부 조건을 잘 구현해주면 되는 문제이다.

    - 수를 정렬할 때 0은 무시하기
    - 등장 횟수 -> 수가 커지는 순으로 정렬하기
    - 행/열 크기가 100을 초과하면 100개 뒤는 버리기
    - 최대 행/열 크기만큼 0 패딩해주기

2. C 연산의 경우 `board`를 transpose해주고 R 연산을 한 후 다시 transpose 해주면 구현이 깔끔해진다.
