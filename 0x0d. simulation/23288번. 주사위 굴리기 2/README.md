# 23288번. 주사위 굴리기 2

## 문제
https://www.acmicpc.net/problem/23288

---

## Key Points

1. 문제에서 요구하는대로 구현하면 되는 문제이다.

    - 주사위를 굴리는 것은 [14499번. 주사위 굴리기](../14499번.%20주사위%20굴리기/) 문제와 동일하게 구현하면 된다.

    - 점수 계산은 bfs를 통해 C를 구할 수 있다.

2. 점수를 미리 계산해두면 조금 더 효율적이다.

    - 문제 순서대로 구현하다보면 점수 계산 시 매번 bfs를 호출하도록 구현하기 쉽다.

    - 하지만, 잘 생각해보면 점수는 사전에 주어진 `board`에 따라 미리 결정되어 있다.
        - B, C는 주사위 굴리기와 무관하게 `board` 만으로 구할 수 있다.
    
    - 즉, `score_board`를 bfs를 통해 사전에 계산해두면, 주사위를 움직일 때마다 bfs를 호출하지 않아도 되어 더욱 효율적이다.
