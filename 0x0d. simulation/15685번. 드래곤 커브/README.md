# 15685번. 드래곤 커브

## 문제
https://www.acmicpc.net/problem/15685

---

## Key Points

1. 이 문제는 아래와 같이 나눠 생각하면 좋다.

    - (0, 0) 시작 기준 d방향 시작 g세대 드래곤 커브 좌표 구하기

    - 위에서 구한 드래곤 커브 좌표를 시작점 위치에 따라 조정하여 board에 표시하기

    - board에서 4개의 꼭지점이 모두 드래곤 커브인 정사각형 개수 세기 => for문으로 세면 됨

2. (0, 0) 기준 d방향 시작 g세대 드래곤 커브 구하는 법

    - d = 0 기준 g세대 드래곤 커브를 구한다.
        - g-1세대 드래곤 커브를 끝 점 기준으로 시계방향으로 회전하여 붙인다.
        - 회전을 쉽게 하기 위해, 끝점을 (0, 0)으로 평행 이동 시키기 -> 회전하기 -> 끝점으로 평행 이동 시키기

    - d만큼 반시계 방향으로 회전시킨다.

3. 이 문제는 x, y 좌표가 직교 좌표계 기준이라 헷갈린다. 자신만의 기준을 두어, 그 기준대로 구현하는 것이 좋은 것 같다.

    - 나는 지금까지 해오던대로 (x, y): (행, 열)을 기준으로 하여 구현하였다.

    - 예제 그림이 (열, 행) 기준으로 나와 있어서 디버깅 할 때 board 출력 시 행과 열을 뒤집어 줘야 하는 불편함이 있었다.

4. 코드를 작성하다보면 뭔가 중복되는 느낌이 들 것이다.

    - g세대 드래곤 커브를 구하는데, g-1세대 드래곤 커브가 필요하다.

    - d = i 인 드래곤 커브를 구하는데, d = 0 인 드래곤 커브가 필요하다.

    - 따라서, `memo_curves[(g, d)]` 딕셔너리로 이전 과정에서 구한 드래곤 커브를 기록해놓으면, 다시 드래곤 커브를 구할 필요 없이 꺼내 쓸 수 있다. (메모이제이션)
