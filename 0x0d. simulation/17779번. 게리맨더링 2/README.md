# 17779번. 게리맨더링 2

## 문제
https://www.acmicpc.net/problem/17779

---

## Key Points

1. 이 문제는 아래와 같이 나누어 생각하면 좋다.

   - 가능한 (x, y, d1, d2) 조합 뽑기

   - 경계선 그리기

   - 1~4번 구역 인구 구하기 -> 5번 구역 인구 구하기
      - 5번 구역 인구 = 전체 - 1~4번 구역 인구

    - min(max(구역 인구) - min(구역 인구)) 구하기

2. 인덱스 계산을 섬세하게 해야하는 문제이다. 실수하기 쉽다.
