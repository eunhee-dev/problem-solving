# 19237번. 어른 상어

## 문제
https://www.acmicpc.net/problem/19237

---

## Key Points

1. 문제에서 구현하라는대로 구현하면 되는 문제이다. 아래와 같이 나누어 생각하면 좋다.

    - 전처리: 상어 딕셔너리 만들기, board 만들기
        - `sharks: {num: ((x, y), d)}`, num: 상어 번호 / (x, y): 좌표 / d: 방향
        - `board: [[(num, scent) for _ in range(n) ] for _ in range(n)]`, num: 상어 번호 / scent: 잔여 냄새

    - 다음 상어 좌표 구하기 (냄새 감소 전에 구해야 함)

    - 냄새 감소시키기 (-1)

    - 상어 이동시키기 (냄새를 감소시키고 이동시켜야 함)
        - 좌표 충돌하면 상어 지우기 (`sharks`를 오름차순으로 만들면 편함)

2. 냄새를 감소시키는 타이밍에 유의하여야 한다.

    - 다음 상어 좌표는 냄새 감소 전에, 실제 상어 이동은 냄새 감소 후에 해야 한다.
