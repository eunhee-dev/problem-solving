# 20056번. 마법사 상어와 파이어볼

## 문제
https://www.acmicpc.net/problem/20056

---

## Key Points

1. 문제에서 구현하라는대로 구현하면 되는 문제이다.

    - 파이어볼 움직이기" 시작 행/열과 끝 행/열이 이어져있는 것은 `% n`으로 처리

    - 같은 좌표에 2개 이상 파이어볼이 있으면 문제 조건대로 합치고 나누기

2. 자료구조 선택 팁

    - 파이어볼이 생겼다가 사라졌다가 함 => `dict`를 쓰면 편하겠구나

    - 좌표에 있는 파이어 볼만 보면 됨 => board를 `defaultdict`로 정의하면 효율적이겠구나

3. 본 문제에서는 이동 후 좌표를 구할 때 `% n`을 하기 때문에 1-based 좌표를 그대로 받아도 문제가 없지만, (다른 문제들과의) 일관성을 유지하기 위해 0-based로 바꾸었음. (실수를 줄일 수 있다)
