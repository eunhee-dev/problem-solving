# 1267번. 핸드폰 요금

## 문제
https://www.acmicpc.net/problem/1267

---

## Key Points

1. 영식(Y): 30초마다 10원 → `(초 // 30 + 1) * 10`


2. 민식(M): 60초마다 15원 → `(초 // 60 + 1) * 15`


3. 문제에서 "29초 또는 그보다 적은 시간 = 10원, 30초~59초 = 20원"이므로, 30초가 되는 순간 다음 요금 구간으로 넘어간다.

    - 즉 0초부터 시작하는 구간이므로 `몫 + 1`로 구간 번호를 구한다. (`// + 1`)


4. 요금이 같으면 `Y M {요금}` 형식으로 둘 다 출력한다.
