# 0x0e. 정렬 I

## 개요

- 기본적인 정렬 알고리즘(버블, 선택, 삽입, 병합, 퀵)의 원리를 이해하고 구현한다.

- Python에서는 `sorted()` 또는 `list.sort()`로 O(n log n) 정렬이 가능하다. (Timsort)

- 카운팅 정렬은 특정 범위 내 정수에 대해 O(n)에 정렬 가능하다.

## 핵심 포인트

<details>
<summary>펼쳐 보기 (문제 풀이 스포일러 포함)</summary>

- **Python 정렬**: `sorted()`는 새 리스트 반환, `list.sort()`는 in-place 정렬.

- **안정 정렬**: 같은 값의 원래 순서가 유지되는 정렬 (병합, Timsort).

- **key 함수**: `sorted(arr, key=lambda x: (x[1], x[0]))`처럼 정렬 기준 지정.

- **카운팅 정렬**: 값의 범위가 작을 때 유용 (수 정렬하기 3).

</details>
