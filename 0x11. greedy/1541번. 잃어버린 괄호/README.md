# 1541번. 잃어버린 괄호

## 문제
https://www.acmicpc.net/problem/1541

---

## Key Points

1. 식의 값을 최소로 만들기 위해서는 - 뒤에 나오는 값을 가장 크게 만들면 된다.

2. 즉, - 뒤에 나오는 값을 전부 빼버리면 값이 최소가 된다.

    - 100 - 50 + 40 - 20 + 30 이라고 하면, 100 - (50 + 40) - (20 + 30) = 100 - 50 - 40 - 20 - 30이 된다.

    - 이와 같이 분배법칙에 의해 적당히 괄호를 치면 - 뒤에 나오는 값을 전부 빼버릴 수 있다.
