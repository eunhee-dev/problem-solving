# 1744번. 수 묶기

## 문제
https://www.acmicpc.net/problem/1744

---

## Key Points

1. 합이 최대가 되기 위해서는 0을 포함한 음수끼리 묶고, 1을 제외한 양수끼리 묶어야 한다.

    - 양수는 큰 수 순으로, 음수는 작은 수 순으로 묶어야 한다.

    - 0은 음수와 묶어 음수를 없애주는 것이 좋다.

    - 1은 묶어서 곱셈으로 처리하는 것보다 덧셈으로 처리하는 것이 좋다.

    - 수학적으로 표현하면, 정수 a, b에 대해 $ab >= a+b$ 인 조건은 $(a-1)(b-1) >= 1$인 경우이다.
