# 15903번. 카드 합체 놀이

## 문제
https://www.acmicpc.net/problem/15903

---

## Key Points

1. 선택한 두 카드의 합이 두 장으로 복사되기 때문에, 마지막에 카드 쌍의 합이 최소가 되기 위해서는 가장 작은 카드 두 개를 항상 고르면 된다. (이 로직을 직관적으로 구현한 풀이, [풀이 - solve_naive.py](./solve_naive.py))

2. 우선 순위 큐를 사용하면 더 효율적인 구현이 가능하다. (정석 풀이, [풀이 - solve.py](./solve.py)) 

    - 가장 작은 카드 두 개를 효율적으로 고르는 방법은, 카드 쌍을 우선순위 큐 형태로 가지고 있는 것이다.
