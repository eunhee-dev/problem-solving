# 1439번. 뒤집기

## 문제
https://www.acmicpc.net/problem/1439

---

## Key Points

1. 0 뭉치의 개수와 1 뭉치의 개수를 세서 더 작은 뭉치의 개수를 뒤집는 것이 최소 횟수이다.

    - [solve.py](./solve.py): `split()`을 사용하여 직관적으로 0과 1뭉치 개수를 센다.

    - [solve_2.py](./solve_2.py): 숫자가 바뀌는 지점을 세어 전체 뭉치의 개수를 세고 반으로 나누어 버린다
        - 숫자가 0/1 두 가지 밖에 없어, 0 뭉치와 1 뭉치는 최대 1개 차이나기 때문
        - 버림으로써 더 작은 횟수를 선택
