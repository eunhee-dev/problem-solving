# 11501번. 주식

## 문제
https://www.acmicpc.net/problem/11501

---

## Key Points

1. 현재 주가를 기준으로 뒤에 더 높은 주가 중 최대 값에서 주식을 팔면 이익이 최대가 된다.

2. 주가를 뒤에서부터 보면서, `max_price`가 현재 주가보다 낮으면 현재 주가로 업데이트하고, 크면 그 가격 차 만큼 이익에 추가 하면 된다.

    - 주가를 앞에서부터 보면, 뒤에 나올 최대 주가를 알아내는데 더 큰 시간복잡도가 필요하다.

    - 뒤에서부터 보면 지금까지 본 것 중 최대값만 유지하면 되기 때문에 그리디하게 주식을 팔 수 있다.

3. python이기 때문에 총 이익의 integer overflow는 고려하지 않아도 된다.
