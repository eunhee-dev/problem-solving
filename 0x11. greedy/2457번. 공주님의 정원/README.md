# 2457번. 공주님의 정원

## 문제
https://www.acmicpc.net/problem/2457

---

## Key Points

1. [1931번. 회의실 배정](../1931번.%20회의실%20배정/) 문제와 유사한 논리로 그리디하게 풀이 가능하다.

    - 꽃이 피는 날짜 -> 꽃이 지는 날짜 순으로 오름차순으로 정렬

    - 3월 1일부터 시작하여, 이 날짜 또는 이전에 피는 꽃들 중 **가장 늦게 지는 꽃**을 선택하면 된다.
        - 최소한으로 선택해야하기 때문에 최대 배정을 목표로 하는 회의실 배정 문제와 반대이다.
        - 또, 회의실 배정 문제와 달리 빈틈이 없어야 한다는 차이점이 있다.
        - 기간에 공백이 생기지 않게하는 꽃을 찾을 수 없으면 0을 출력한다.

    - 선택된 꽃이 지는 날을 기준으로, 다시 그날 이전에 피는 꽃들 중에서 가장 늦게 지는 꽃을 선택하는 과정을 반복한다.

    - 이 과정을 11월 30일까지 반복한다.

2. 날짜를 일자로 변환하면 비교가 더 수월하다.

    - 년도를 고려하지 않기 때문에, 월 -> 일로 바꾸는 구현만 하면 된다.

    - 누적합을 활용하면 조금 더 효율적이다.
