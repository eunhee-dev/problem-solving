# 1026번. 보물

## 문제
https://www.acmicpc.net/problem/1026

---

## Key Points

1. A의 작은 값부터 차례대로 B의 큰 수에 매칭하면 최소 값이 된다.

    - [재배열 부등식](https://namu.wiki/w/재배열%20부등식)으로 증명 가능하다.

    - 재배열 부등식을 모르더라도, 직감적으로 풀이를 떠올려 시도해 볼 수 있다.

    - 완전 탐색은 O(N! * N)이 걸리므로 N <= 50인 현재 문제에서는 풀이 불가능하다.

2. 문제 조건에서는 B의 순서를 변경하지 말라고 했지만, 구현 상으로는 B도 정렬하면 구현이 편리하다.
