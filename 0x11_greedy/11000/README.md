# 11000번. 강의실 배정

## 문제
https://www.acmicpc.net/problem/11000

---

## Key Points

1. 수업이 최대한 많이 겹치는 시기의 강의 수를 구하면 되는 문제이다. [풀이 - solve.py](./solve.py)

    - 수업의 시작/종료를 이벤트(시간, state)로 저장한다.
        - 수업 시작 state: +1
        - 수업 종료 state: -1

    - 시간 => state 순으로 정렬한다.
        - 종료 state를 -1로 설정했기 때문에 오름차순으로 정렬해야 수업이 종료된 후 곧바로 다른 수업을 시작할 수 있다.

    - 시간 별 최대 강의 수를 확인하면 그 만큼 강의실을 준비하면 된다.

2. 우선순위 큐로도 풀이 가능하다. [풀이 - solve_heapq.py](./solve_heapq.py)

    - 수업 시작 순으로 `time_table`을 정렬한다.

    - 현재 사용 중인 강의실들의 종료 시간을 우선순위 큐로 저장한다.

    - 우선순위 큐 특성상 가장 빨리 끝나는 강의실을 O(1)에 조회할 수 있고, 강의실 배정 및 갱신을 O(logN)에 수행할 수 있다.
