# 6549번. 히스토그램에서 가장 큰 직사각형

## 문제
https://www.acmicpc.net/problem/6549

---

## Key Points

1. 오작수를 구하는 문제이다.
    - 스택을 오름차순으로 유지
    - 작은 사각형이 나오면, 더 큰 사각형들을 스택에서 모두 제거
    - width 계산 (스택 top의 index ~ 오작수 index 거리, 스택이 비어있으면 제일 앞까지의 거리)
    - 넓이 계산 (max 값 저장)

2. 오작수가 없는 값들에 대해서는 끝(n)을 기준으로 계산한다.

3. [6198번. 옥상 정원 꾸미기](../6198/)와 비슷한 느낌이지만 거리 계산에서 차이가 있다.

    - 6198번 문제는 오른쪽만 보기 때문에 `width = idx - prev_idx - 1`

    - 본 문제에서는 왼쪽까지 봐야하기 때문에, `width = idx - stack[-1] - 1`
